# Community Skills Directory

Entries in this directory are valid community submissions that are not synced/exported to ClawHub tooling.

Store non-synced community entries at:

- skills/community/<category>/<entry-slug>.md

Category roots (folder names):

- strategies
- recipes
- workflows
- skills
- filesystem

Frontmatter `type` values remain singular:

- strategy
- recipe
- workflow
- skill
- filesystem

Maintainers can promote entries from `skills/community/*` to `skills/official/*` when ready.
