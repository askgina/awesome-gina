# Official Skills Directory

Entries in this directory are the official set synced/exported to ClawHub tooling.

Store synced community entries at:

- skills/official/<category>/<entry-slug>.md

Category roots (folder names):

- strategies
- recipes
- workflows
- skills
- filesystem

Frontmatter `type` values remain singular:

- strategy
- recipe
- workflow
- skill
- filesystem

For non-synced community submissions, use `skills/community/*`.
